# JustStreamIt

## Prérequis
Pour exécuter ce projet, vous devez avoir les outils suivants installés :
- **Python** : Version 3.8 ou supérieure.
- **Git** : Outil de contrôle de version pour cloner le dépaôt.
- **Terminal** : Un terminal comme Command Prompt (Windows), Terminal (macOS), ou un shell Linux.


## Installer et lancer l'API

### 1. Cloner le dépôt Github de l'API en local
Dans votre terminal, tapez :
```bash
git clone https://github.com/OpenClassrooms-Student-Center/OCMovies-API-EN-FR.git
```

### 2. Suivez les étapes du fichier README de celui-ci


## Installer et lancer JustStreamIt

### 1. Cloner ce dépôt Github en local
Dans votre terminal, tapez :
```bash
git clone https://github.com/redaabdi/P6-Developpez-une-interface-utilisateur-pour-une-application-web-Python
```


### 2. Ouvrez avec un navigateur internet le fichier "index.html"
